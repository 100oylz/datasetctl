# datasetctl 🚀

`datasetctl` 是一个高性能、轻量级的数据集管理命令行工具，专为数据科学家和 ML 工程师设计。它通过 **符号链接（Symlinks）** 机制，实现原始数据集与用户工作区之间的解耦，避免了海量数据的重复复制。

## ✨ 核心特性

- **标准化管理**：统一管理原始数据（Raw）与预处理方案（Processed）。
- **零拷贝引用**：利用 Linux 软链接技术，瞬间为用户工作区提供数据访问。
- **自动化路径**：默认管理路径为 `~/data/dataset`，自动处理用户主目录展开。
- **安全健壮**：基于 Rust 编写，严格的错误检查与幂等性操作。

## 📂 目录结构规范

工具会自动建立并维护以下层级结构：
```text
~/data/dataset/
└── [dataset_name]/
    ├── rawdata/          # 原始、不可变数据
    └── processeddata/    # 各种预处理后的方案版本
        └── [scheme_name]/
            ├── data/     # 处理后的数据
            └── README.md # 方案说明文档
